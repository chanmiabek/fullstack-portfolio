About Me

My Journey

As a fullstack developer, Software developer and web designer, I've transformed challenges into opportunities for growth and innovation. My journey in tech began with a curiosity about how digital solutions could address the unique challenges faced in refugee settings.

Despite limited resources, I taught myself programming through online courses, community workshops, and countless hours of practice. What started as a personal pursuit has evolved into a mission to leverage technology for positive change.

Today, I specialize in creating responsive web applications, software solutions, and community-focused digital tools that bridge gaps and create opportunities for underserved communities.

My Mission

Empowering Through Technology

To use my skills to create digital solutions that address real-world challenges faced by refugee and marginalized communities.

Inspiring Others

To demonstrate that circumstances don't define potential, and to mentor aspiring developers from similar backgrounds.

Building Bridges

To connect refugee talent with global opportunities through technology and collaborative projects.

A popular JavaScript library for building user interfaces efficiently. Facilitates the creation of reusable components and single-page applications.

Bootstrap CSS

A widely-used, open-source CSS framework for developing responsive and mobile-first projects on the web. Provides pre-designed components and a grid system.
